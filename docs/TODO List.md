# Список задач

## Витрина промптов (супергруппа)

- Пользователь не зарегистрирован в боте. Нажимает кнопку скопировать промпт себе или открыть в редакторе. Происходит регистрация пользователя в боте. В поле fsm_data добавляется пометка, что пользователь не заходил в личный чат с ботом. Для него записывеется выбранный промпт и выводится сообщение (решить как), где сообщение и как туда попасть. Этои промпты будут выведены в чат, когда пользователь стартонет его.
