# Формат шаблонов
Шаблон описывает форму, которая выводится на странице для заполнения промпта и формат вывода JSON промпта (ответа).
Шаблон описывается в формате JSON.
Спецификация шаблона: docs/frontend/create_form/example_json_sample.md.

## Служебные поля
Шаблон использует так-же поля не выводимые в JSON ответе. Которые служат для построения HTML формы и управления формированием ответа:
- _type - тип поля
- _label - описание поля
- _options - варианты выбора
- _value - значение выбора
- _title - заголовок
- _spoiler - заголовок в спойлере
- _important - наличие поля указывает что поле или объект обязательно должно быть заполнено для включения родительского элемента в ответ. 

## Типы полей
- readonly - только для чтения
- text - текстовое поле
- number - цифровое поле
- textarea - многострочное текстовое поле
- color - поле для выбора цвета
- select - выпадающий список
- checkbox - множественный выбор
- box - блок
- array - массив
- object - объект массива

## Обязательные поля в корне шаблона
Обязательные поля должны находиться в корне json шаблона.

### version
- Версия шаблона в шаблоне.
- В промпте версия шаблона по которому построен этот промпт. 
Служит для выбора правильного шаблона при построении формы.  
Пример:
  ```
  "version": "video-1.0 (не учитывать в видео)"
  ```

- Первая часть, до тире - указывает на совместимость промпта и шаблона.
- Вторая часть, цифровая - это сама версия. 
- Через пробел может содержать комментарий для модели.

### title
Название и краткое описание промпта.
Пример:
  ```
"title": "Восход солнца над морем."
  ```
