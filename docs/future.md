## Доработка формы

### Работа с массивами
Реализация возможности полного контроля над объектами массивов: перемещение, дублирование и дт.  
Для смены id объекта нужно:
- data-path, id, for, data-parent, data-target обновить во всех объектах. Добавить в объекты data-label с любым значением, чтобы по нему найти названия и поменять цифры, найти data-id у объектов и заменить их.
